How to set up Firebase (free)
Step by step — takes 5 minutes
▼
  1. Go to console.firebase.google.com and sign in with Google
  2. Click Add project → name it brand-rehab → click through the setup
  3. In your project, click Build → Realtime Database → Create database
  4. Choose Start in test mode → click Enable
  5. Go to Project Settings (gear icon) → scroll to "Your apps" → click the web icon </>
  6. Register your app → copy the apiKey and projectId values
  7. Your Database URL is in the Realtime Database section — looks like https://yourproject-default-rtdb.firebaseio.com
  8. Paste those values in the fields above → click Save
