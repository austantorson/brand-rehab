Setup Guide
How to create your free Firebase project
▼
  1. Go to console.firebase.google.com → sign in with Google
  2. Click Add project → name it brand-rehab
  3. Build → Realtime Database → Create database → Start in test mode
  4. Go to Project Settings (gear icon) → scroll to "Your apps" → click </>
  5. Register your app → copy apiKey and projectId
  6. Your Database URL is in the Realtime Database panel — format: https://yourproject-default-rtdb.firebaseio.com
  7. Paste all three values above → Save & Connect
